This program launches server which is returning number of special case, uppercase, lowercase letters and numbers written in querry.
Querry should be written as "localhost:8080/content/{yoursString}".
JSON returned:
    "string" - Yours string
    "numberCounter" - How many numbers are in string
    "lowerCaseCounter" - How many lower case letters are in string
    "upperCaseCounter" - How many upper case letters are in string
    "specialCaseCounter" - How many special case letters are in string
